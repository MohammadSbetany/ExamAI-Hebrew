# ExamAI-Hebrew

Automatic generation of open-ended questions in Hebrew from learning materials.

---

## About the Project
ExamAI-Hebrew is a software engineering project aimed at building a system that
automatically generates open-ended questions in Hebrew from learning materials.
The system is designed to support both teaching and self-learning processes.

---

## MVP Scope
- Upload a PDF file
- Extract Hebrew text from the file
- Generate open-ended questions
- Display the generated questions

---

## Technologies
- Frontend: React
- Backend: Python (FastAPI)
